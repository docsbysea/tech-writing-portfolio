Step 1: Create a New Repository

  1. Sign in to your GitHub account.
  2. Click the + icon in the top-right corner and select New repository.
  3. Enter a repository name (e.g., my-blog).
  4. Choose Public (so others can view your blog) or Private (for restricted access).
  5. Initialize the repository with a README.md file (this will contain the content of your blog).
  6. Click Create repository.
    Your repository is now ready for adding files.
