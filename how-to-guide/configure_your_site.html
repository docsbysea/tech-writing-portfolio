Step 2: Configure Your Site

  1. In your new repository, click Add file > Create new file.
  2. Name the file _config.yml.
  3. Click on Edit this file in the upper-right corner.
  4. Add the following basic configuration settings: 
    title: My Blog
description: This is my personal blog.
theme: jekyll-theme-minimal
  5. Browse the GitHub Pages theme library to choose a theme. Replace jekyll-theme-minimal with your preferred theme name in _config.yml. You can also customize your own theme.
  6. Click Commit new file to save your changes.

    Your static site is now configured with the selected theme.
